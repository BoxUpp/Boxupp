*********************************************************************************
*Pre-requisites for getting Boxupp up and running:                              *
*********************************************************************************

1. Oracle Virtualbox - a powerful x86 and AMD64/Intel64 virtualization product

2. Vagrant - a free and open-source software for creating and configuring virtual and repeatable development environments

3. Java Runtime Environment (JRE 7.0+)- a runtime environment for running Java applications

4. Vagrant docker plugin - If you want to create Containers on your machine


If something doesn't seem to work as expected, please get in touch with us at support@boxupp.com
